Graph

global interface Graph
DescriptionInterface to specify a Graph ADT. A graph is a set of vertices and a set of edges. Vertices are represented by integers from 0 to n - 1. Edges are ordered pairs of vertices. Each implementation of the Graph interface should provide a constructor that specifies the number of vertices and whether or not the graph is directed.
